This project was implemented using Python 2.7 and using Anaconda 2 Sypder.

Sklearn
In regards sklearn library version, the sklearn library was updated using the instructions from this link:
http://scikit-learn.org/stable/install.html

XGBoost
For the XGBoost library, the XGBoost library was configured using the instructions from the link below. It will require the installation of the mingw application.
https://www.ibm.com/developerworks/community/blogs/jfp/entry/Installing_XGBoost_For_Anaconda_on_Windows?lang=en


Data
The data has been provided and included in every folder

How to run the project.
1.  For the files in the 2, 2b, 2d, 3a folders, no additional setup is necessary.

2a.  For the "3b - 4a" folder, the mingw_path variable needs to be adjusted for the following python files:

* 3b - Supporting Functions (line 8)
* 3b,c - Implementation & Refinement (line 159)
* 3d,4a - Refinement & Free-Form Visualization (line 63)

2b.  For the "3b - 4a" folder, make sure that you run the "3b-Supporting Functions" python file first before running the other python files.
